# Startup Funding Analysis Using SQL

## Overview

This project explores startup funding data through SQL-based Exploratory Data Analysis (EDA). The goal was to analyze funding trends, sector performance, investor activity, startup concentration, and funding distribution patterns while applying advanced SQL techniques to generate business-oriented insights.

The project focuses on transforming raw funding data into meaningful findings using aggregations, window functions, ranking techniques, statistical measures, and trend analysis.

Note: This project uses a sample dataset for educational and portfolio purposes. The objective is to demonstrate SQL analysis skills, analytical thinking, and business insight generation.

---

## Business Questions

This analysis aims to answer the following questions:

* How has startup funding changed over time?
* Which sectors receive the highest levels of investment?
* Which cities attract the most startup funding?
* How concentrated is funding among startups?
* How concentrated is investor participation?
* Is startup funding distributed evenly or concentrated among a small number of companies?
* How do funding patterns vary across different funding stages?

## Key Findings

### Funding Trends

Funding activity exhibited noticeable variation across years, indicating changing investment conditions and market sentiment.

### Funding Concentration

A relatively small number of startups accounted for a large share of total funding, suggesting significant concentration of capital.

### Investor Activity

Investment participation was concentrated among a limited group of active investors.

### Sector Performance

Certain sectors consistently attracted higher levels of investment, highlighting investor preferences within the startup ecosystem.

### Geographic Concentration

Startup funding was concentrated within a small number of major startup hubs.

### Funding Distribution

The difference between average and median funding values indicated the presence of large funding outliers and an uneven distribution of capital.
